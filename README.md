## Requirements

- Godot 4.4.1 or later
- Python 3.8 or later
- CMake 3.13 or later (for building dependencies)
- C++17 compatible compiler
- Git

## Quick Start
```bash
# Clone repository
git clone https://github.com/RectangleEquals/godotai.git
cd godotai

# Initialize project
python setup.py
# Choose: init

# Build extension
python setup.py
# Choose: build

# Install to Godot project
python setup.py
# Choose: install
```
